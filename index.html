Using Git & GitHub:

Download the github repo from github using cmd, by the following commands:

  1. Download git.
  2. To clone the whole repository: 
    git clone https://github.com/ansus6776576576/repository.git

To download a specific file, use 'curl' or 'wget':

  1. Using curl: 
    curl -L -o filename.ext https://raw.githubusercontent.com/ansus/788787878/repository/branch/path/to/file
  2. Using wget: 
    wget -O filename.ext https://raw.githubusercontent.com/repository/branch/path/to/file
